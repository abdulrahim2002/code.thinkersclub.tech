Remote-Code-Command-execution

A Remote Code/Command execution client, where client machine can securely execute code remotely on server machine. Features live screen sharing, along with user authentication and OTP. Used TCP/IP sockets in python to exchange data
git clone https://github.com/abdulrahim2002/Remote-Code-Command-execution
